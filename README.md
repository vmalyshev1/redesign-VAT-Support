# VAT.SUPPORT - Precision Finance Redesign

Modern teal-themed redesign for vat.support WordPress site (Picostrap5 theme).

## Quick Start

1. Upload `precision-finance-override.css` to `/wp-content/themes/vat-child/`
2. Add the code from `functions-snippet.php` to your child theme's `functions.php`
3. Clear cache and refresh

## Files

| File | Description |
|------|-------------|
| `precision-finance-override.css` | Main stylesheet with all new styles |
| `functions-snippet.php` | PHP code to load the styles (priority 9999) |
| `demo.html` | Full preview of the redesign |
| `README-INSTALLATION.md` | Detailed installation guide |
| `html-templates/` | HTML templates for each section |

## Preview

Open `demo.html` in your browser to see the full redesign.

## Color Palette

- Primary: `#0d9488` (Teal 600)
- Accent: `#14b8a6` (Teal 500)
- Dark: `#0f172a` (Slate 900)

## Typography

- Headings: DM Sans (600 weight)
- Italic accents: DM Serif Display
